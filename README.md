# Diwali_Sales_Data_Analysis-Python_Pandas_EDA

Performed end-to-end exploratory data analysis (EDA) on Diwali sales data using Python (Pandas, NumPy, Matplotlib/Seaborn) to understand customer purchasing behavior and festive demand trends.

Cleaned and preprocessed raw datasets by handling missing values, data type inconsistencies, and outliers, improving data reliability for analysis.

Analyzed customer demographics (gender, age group, marital status) to identify high-spending customer segments during the festive season.

Conducted category-wise and product-wise sales analysis, highlighting top-performing product categories contributing to overall revenue.

Generated visual insights to support data-driven marketing and inventory planning decisions for festive sales campaigns.
